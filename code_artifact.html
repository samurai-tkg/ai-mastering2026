午前：プロンプト設計とビジュアル制作の極意

午前中の研修の目的は、「ただAIにお願いする」曖昧な態度を脱し、プロとしての出力制約とデザイン基準を手に入れることです。 以下の「プロンプトビルダー」を使用して、精緻な役割定義と出力フォーマットを強制した指示を作成・実験してください。

構造化プロンプト・ビルダー

穴埋めするだけで、AIが最も理解しやすい指示文を生成します

デザインの黄金三原則

  • 1
    定規・ガイドラインを常時ONに

    Canvaでは「ファイル設定」＞「定規とガイドを表示」に。要素の「端」を整列させるだけで、アマチュア感が完全に消えます。

  • 2
    ジャンプ率（文字サイズ差）の強調

    最も読ませたいキャッチコピーは「極太・最大」にし、補足の文字は「極細・最小」に。メリハリこそが洗練のデザインです。

  • 3
    背景とのコントラストを最優先

    どれだけ綺麗な色でも、文字が背景に溶け込むと情報価値はゼロです。白背景には深黒、暗背景には極白が基本です。

カラーコントラスト・チェック基準

WCAG AA 規格 (標準)

文字サイズが小さい場合の視認性最低基準

比率 4.5:1
WCAG AAA 規格 (最高解像)

どんなディスプレイ・環境でも確実に見やすい基準

比率 7:1

※無料版Canvaでは「RGB」カラープロファイルで書き出されます。画面上でコントラストがギリギリだと、端末（モバイル等）ごとに見え方が大きく異なります。

音声・テロップ完全同期（ミリ秒単位）の技術概念

Appendix（付録）：ビジネスへの展開と自動化システムアーキテクチャ

本日の本編技術をマスターした後、実際に商業ビジネスや社内受託インフラとして展開する際の、 「外部APIや決済（PayPayなど）、音声クローン（ElevenLabs）、公式LINE、高精細商業印刷」の接続仕様をここに解説します。

1. LINE公式＆クラウド受電自動化

IP電話(Twilio)と文字起こし(Whisper)を連携し、顧客からの着信通話を自動録音・即時テキスト化して公式LINE宛てに要約通知を飛ばします。

[着信] ──> Twilio SIP
[録音] ──> OpenAI Whisper API
[要約] ──> Gemini-2.5-Flash
[配信] ──> LINE Webhook (yoom経由)

2. 店長の「音声デジタルクローン」技術

ElevenLabsのクローン生成モデルを用いて店長自身の声色をデジタルクローン化。飲食店の紹介やSNS動画にその声をあて、制作コストを削減。

🚨 注意事項と商用展開ルール： 無料プランでは商用動画（広告・収益化等）の利用はライセンス規約違反です。商業利用は最低でもStarterプラン契約が必要。必ず本人の声紋認証同意書が必要です。

3. Canva Pro 商業印刷完全データ(300dpi/CMYK)

Canvaの有償プランにより、印刷所へそのまま入稿して色ブレや文字化けを起こさない高解像度PDFデータの出力を制御します。

RGB(画面用)から CMYKカラープロファイル へ変更出力
透明度不具合を回避する PDFの平坦化（Flattening）
3mmの「塗り足し（裁ち落とし）」とトリムマークを自動生成

4. LINE上のPayPay決済と自動Webhook連携

顧客がその場でスピーディに決済し、自動で制作本番プロセスをキックするための仕組み。

⚠ 規約違反リスクの回避： 個人の送金リンクやQRコードを商業売上に使うのはPayPay規約上厳禁（即座に口座凍結されます）。必ず「PayPay for Business（正規加盟店）」のAPIを利用して自動決済URLを生成させます。
