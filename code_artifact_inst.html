03

午前：プロンプト設計における「制御因子」の厳密化

AIから思い通りの成果を引き出すための基本は、曖昧な指示文の徹底排除です。 ターゲット、制約条件、そして特に出力フォーマットの形式固定を規定します。

ROLE (役割の定義) プロのコピーライター

ただ記述させるのではなく、視座や専門知識を指定して出力品質のバイアスを上げます。

TARGET (狙うセグメント) 20代〜30代の働く女性

特定層のペルソナへ絞ることで、言葉の選定・トーンをAIが適合させます。

FORMAT (出力形式の強制) 30文字、5件

文字数制限、表形式、JSONなどの構造化を指定し、そのまま流用可能にします。

CONSTRAINTS (制約事項) ハッシュタグは3つ

「専門用語を省く」「平易な口語体に」など除外定義を設定し、ズレを防ぎます。

04

午前：Canvaをデザインのルールで縛る

無料テンプレートをただ「なんとなく」配置すると、かえって素人っぽさ（アマチュア感）が強調されてしまいます。 講師から伝える、絶対に守るべき「整列と対比」の基本3箇条です。

PRINCIPLE 1

定規（ガイド）の一致

Canvaのファイル設定から「定規とガイド」を有効にし、テキスト、枠線、画像の角など「外側の線」を1ピクセルの狂いもなく揃えます。

PRINCIPLE 2

極端なジャンプ率（強弱）

メインコピーは「特大・極太」、補足情報は「極小・細」にして明快な主従関係を作ります。中途半端なサイズ差は「ただのズレ」に見えてしまいます。

PRINCIPLE 3

コントラスト比の確保

文字色が背景色と同化するのを避けます。WCAG AA規格である「4.5:1」をクリアさせ、可読性を最大化します。

デザイン整列チェック

【Canva整列・強弱ルールの確認】

・ファイル設定から「定規とガイドライン」を有効化し、テキストや図の端を1ミリのズレもなく揃えましたか？

・文字色と背景のカラーコントラスト比（4.5:1以上）は担保できていますか？クリアできたら「チェック2・3」へ！

05

午後：AI動画生成プロンプトと音声同期の微調整技術

最新の動画生成AIでダイナミックな動きを作るには、英語による的確なカメラコントロール命令が必要です。また、合成音声（TTS）を使用する際は無音部のミリ秒トリミング処理が映像の「プロっぽさ」を左右します。

英語カメラプロンプトパターン

dolly zoom inward 被写体を強調し迫るドリーズーム
360-degree orbit around the subject 被写体の周囲を一周するように旋回撮影
crane shot rising up 垂直に浮き上がる広範囲撮影

音声トリミング＆同期

AI出力音声の直前には、必ず数百ミリ秒の「無音（空白）」が入ります。

  • タイムライン上でテロップ（字幕）出現のまさにその瞬間、音波形が立ち上がるように先頭を「前詰め」カット。
  • 無音部を放置すると、人間の脳は直感的に「非同期的」なアマチュア動画として違和感を覚えます。
タイミング同期チェック

【カメラ制御とミリ秒同期の確認】

・ポータルのジェネレーターを使って、ダイナミックな動き（Dolly, Orbitなど）を持たせた動画を生成できましたか？

・音声の先頭にある無音の空白をカットし、テロップの出現とピッタリ声が重なるように同期できたら「チェック4〜7」へ！

午後：GASソースコード解説の要諦

受講生の混乱を防ぐために、講師は以下の**3大コア・ステートメント（中心コード）**の役割を確実に説明してください。

// キーを安全に環境から引き出す（直書き漏洩防止） const apiKey = PropertiesService.getScriptProperties().getProperty('GEMINI_API_KEY');
// JSON以外の余計なノイズ言葉（「かしこまりました」等）を排除し構造化出力を強制 "generationConfig": { "responseMimeType": "application/json" }
// 返却されたJSONテキストをJavascriptオブジェクトに変換し、各セルへ書き込める状態にする const result = JSON.parse(responseText);

Appendix：商業ビジネスへのスケールアップ構想（付録）

研修後の実業展開に向けて、顧客からのヒアリングから納品・決済までの完全自動システム構成案（ブルー・プリント）を提示します。

Twilio SIPで受電 ──> Whisperで瞬時に録音ファイルを高精度文字起こし ──> Geminiがデザインの構想・カラーを自動要約してLINE通知。

ElevenLabsに店長の本人の声を数十秒学習させ、音声クローンを作成。生成されたテキストを店長本人の声色で、高精度の自動ナレーション動画化。

印刷所用カラープロファイル「CMYK」、解像度「300dpi」、不具合防止の「PDFの平坦化」を指定して、文字崩れ・色ブレのない実用データを取得。

PayPay公式 APIによるURL即時自動決済

個人送金リンクは使用せず、「PayPay for Business」APIを用いて動的に正規決済リンクを作成、LINEチャット上で非対面の瞬時課金・納品。
